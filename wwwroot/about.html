About Leylines Map

The Leylines Map is a website for map-based access to spatial data all about leylines and related information.

The site was completely redesigned and converted to a new framework. Not all catalog entries are available yet, i will add them piece by piece.

This site is a project i develop in my spare time. As a family man and 80% employee it is not always easy to find the time to maintain and develop the map.

The Leylines Map is a fully open architecture. When you access data through it, you are typically accessing the data directly from a data-provider which is the custodian of that data.

To see what data is available on the Leylines Map, refer to the Data Catalogue in the Leylines Map itself. Click the Data Catalogue tab and expand the Data Set group names.

How to use the Leylines Map

Just go to https://maps.leylines.net/. Most operations are obvious, but you can click the Help and FAQs button near the top of this page for specific instructions.

For further information email info@leylines.net. Feel free to send me feedback and bug reports.

Leylines Map Software

The Leylines Map is Open Source software available as a Github project. The core platform, TerriaJS, was initially developed by Data61 for the NationalMap, but has subsequently been used for other projects. It has been made available to assist with the development of other spatial data viewing applications.
